# StudySync - 学習記録アプリケーション

日々の学習時間を科目ごとに記録・可視化するWebアプリケーションです。

## 使用技術
- Node.js
- Express
- EJS
- Prisma (ORM)
- PostgreSQL (Neon DB)
- Tailwind CSS / Chart.js

## ディレクトリ構成
node_modules/ は容量削減のため含めていません。以下の手順でセットアップしてください。

## 起動方法

### 1. 依存ライブラリのインストール
ターミナルでこのフォルダを開き、以下のコマンドを実行してライブラリをインストールしてください。

```bash
npm install
```

### 2. 環境変数の設定
.env.example ファイルをコピーして .env という名前のファイルを作成してください。
作成した .env ファイル内の DATABASE_URL に、PostgreSQLの接続文字列を記述してください。
（※提出先の方へ: 今回使用しているNeon DBの接続情報は、ご自身の環境をご利用ください）

Mac/Linuxの場合
```bash
cp .env.example .env
```

Windowsの場合
```bash
copy .env.example .env
```

### 3. データベースのセットアップ
Prismaを使用してデータベースのスキーマを同期します。
```bash
npx prisma db push
```
⚠️ Prismaのバージョン通知について コマンド実行時にターミナルに Update available 6.19.1 -> 7.2.0 といった更新通知が表示されることがありますが、無視してください。 本アプリケーションは v6.19.1 で動作確認を行っています。v7.0以降へアップデートすると互換性がなくなり正常に動作しなくなるため、アップデートは行わないようにお願いいたします。

### 4. アプリケーションの起動
以下のコマンドでサーバーを起動します。
```bash
node app.js
```

### 5. ブラウザで確認
ブラウザで以下のURLにアクセスしてください。
http://localhost:3000

## 機能概要
- ダッシュボード: 学習時間の合計をグラフで確認できます。
- 科目追加: 任意の科目とテーマカラーを登録できます。
- ログ記録: 科目、学習時間、メモを記録できます。
- 削除機能: 間違えた記録を削除できます。
